# VietWebPro - Dịch Vụ Thiết Kế Website

🚀 Trang giới thiệu dịch vụ thiết kế website chuyên nghiệp kèm 5 demo mẫu.

## Tech Stack

- HTML5, CSS3 (Vanilla), JavaScript
- Google Fonts (Inter, Outfit, Playfair Display, Space Grotesk)
- Responsive Design, Glassmorphism, Dark Theme

## Chạy Local

Cần XAMPP hoặc bất kỳ web server nào. Mở `http://localhost/mywebsite/`.

## Cấu Trúc

```
mywebsite/
├── index.html          # Trang chủ giới thiệu dịch vụ
├── css/style.css       # Design system
├── js/main.js          # Animations & interactions
├── images/             # Thumbnail demo pages
└── demos/
    ├── business/       # Demo doanh nghiệp
    ├── ecommerce/      # Demo thương mại điện tử
    ├── portfolio/      # Demo portfolio
    ├── restaurant/     # Demo nhà hàng
    └── realestate/     # Demo bất động sản
```
